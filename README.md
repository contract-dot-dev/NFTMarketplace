# NFTMarketplace

This repo contains a Foundry project with a `NFTMarketplace` contract.
It’s designed as an example repo for use with [contract.dev](https://contract.dev).

contract.dev gives smart contract developers production-like staging environments for EVM applications.

Each project includes a [Stagenet](https://docs.contract.dev/stagenets) — a private EVM testnet with built-in tools for testing, simulating, and analysing your contracts before mainnet.

## Get Started

Follow our [Quickstart](https://docs.contract.dev/quickstart) to import this repo’s contracts onto a Stagenet and set up analytics [Workspaces](https://docs.contract.dev/analytics/workspaces) for them.

The steps are straightforward. All you need to do is:

1. Spin up a Stagenet
2. Import this repository from GitHub
3. Run its deployment script against your Stagenet
4. Inspect the deployed contract in its dedicated Workspace
